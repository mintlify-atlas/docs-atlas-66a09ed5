Before installing ReVanced Manager, make sure your device meets the following requirements.

## Requirements

**Android 8.0 (Oreo) or higher** — ReVanced Manager requires API level 26 (`minSdk = 26`) or above. Check your Android version in **Settings > About phone**.

**Installation from unknown sources enabled** — ReVanced Manager is distributed as an APK outside the Play Store. Android will prompt you to allow installation from unknown sources when you open the file. You can enable this in **Settings > Apps > Special app access > Install unknown apps**.

**Sufficient storage space** — Patching creates a modified copy of the target app. Make sure you have enough free storage for both the original app and the patched output.
